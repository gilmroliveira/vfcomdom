# Tarefa 6 - Validação de Formulário com DOM

## Descrição do Projeto
Este projeto consiste na criação de um formulário HTML com campos para Nome, E-mail e Telefone, utilizando JavaScript para validar os dados no momento do envio conforme regras definidas na tarefa da disciplina Desenvolvimento Web II.

## Requisitos da Tarefa
- Campo Nome: Deve conter entre 5 e 40 caracteres.
- Campo E-mail: Deve conter o caractere "@".
- Campo Telefone: Deve conter exatamente 11 dígitos, correspondendo a um número de celular com DDD (formato esperado: (##)#-####-####).
- Nenhum campo pode ser deixado vazio. Caso algum campo obrigatório não seja preenchido ou não respeite as regras, o formulário não será enviado e mostrará uma mensagem de erro ao usuário.

## Uso
1. Abra o arquivo `index.html` em um navegador.
2. Preencha os campos com os dados solicitados.
3. Clique no botão "Enviar".
4. Caso algum campo esteja inválido ou vazio, uma mensagem alertará quais erros devem ser corrigidos.
5. Quando todos os dados estiverem válidos, o formulário poderá ser enviado normalmente.

## Equipe
- Nome completo do participante 1
- Nome completo do participante 2 (se houver)
- Nome completo do participante 3 (se houver)
- Nome completo do participante 4 (se houver)
